# Blinkit-Analysis-Excel-Dashboard
Created an interactive sales dashboard in Excel for Blinkit to analyze $1.2M sales across outlets, items, and locations. Used pivot tables, charts, and slicers to track KPIs like avg sales, ratings, fat content, and item type. Gained insights on top categories, outlet size performance, and customer trends.
